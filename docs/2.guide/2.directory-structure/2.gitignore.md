`.gitignore` 文件指定了 Git 应该忽略的有意未被跟踪的文件。

:read-more{icon="i-simple-icons-git" title="Git 文档" to="https://git-scm.com/docs/gitignore" target="_blank"}

我们推荐 `.gitignore` 文件**至少**包含以下内容：

```bash [.gitignore]
# Nuxt 开发/构建输出
.output
.data
.nuxt
.nitro
.cache
dist

# Node 依赖
node_modules

# 日志
logs
*.log

# 杂项
.DS_Store

# 本地环境文件
.env
.env.*
!.env.example
```
