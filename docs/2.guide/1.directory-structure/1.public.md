位于 `public/` 目录中的文件会在站点根路径提供，并且不会在构建过程中被修改。适用于那些必须保留文件名（例如 `robots.txt`）或很可能不会改变（例如 `favicon.ico`）的文件。

```bash [Directory structure]
-| public/
---| favicon.ico
---| og-image.png
---| robots.txt
```

```vue [app/app.vue]
<script setup lang="ts">
useSeoMeta({
  ogImage: '/og-image.png',
})
</script>
```

::tip{to="https://v2.nuxt.com/docs/directory-structure/static" target="_blank"}
在 Nuxt 2 中，这被称为 [`static/`] 目录。
::
