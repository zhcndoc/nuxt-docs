The `nuxt.config` file extension can either be `.js`, `.ts` or `.mjs`.

```ts twoslash [nuxt.config.ts]
export default defineNuxtConfig({
  // 我的 Nuxt 配置
})
```

::tip
`defineNuxtConfig` 辅助函数是全局可用的，无需导入。
::

You can explicitly import `defineNuxtConfig` from `nuxt/config` if you prefer:

```ts twoslash [nuxt.config.ts]
import { defineNuxtConfig } from 'nuxt/config'

export default defineNuxtConfig({
  // 我的 Nuxt 配置
})
```

::read-more{to="/docs/4.x/api/configuration/nuxt-config"}
在 **Nuxt 配置** 文档中了解所有可用选项。
::

为确保你的配置是最新的，Nuxt 在检测到主配置文件、[`.env`](/docs/4.x/guide/directory-structure/env)、[`.nuxtignore`](/docs/4.x/guide/directory-structure/nuxtignore) 和 [`.nuxtrc`](/docs/4.x/guide/directory-structure/nuxtrc) 点文件发生更改时会进行完整重启。
