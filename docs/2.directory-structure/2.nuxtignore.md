`.nuxtignore` 文件告诉 Nuxt 在构建阶段忽略项目根目录中的文件（[`rootDir`](/docs/4.x/api/nuxt-config#rootdir)）。

它遵循与 [`.gitignore`](/docs/4.x/directory-structure/gitignore) 和 `.eslintignore` 文件相同的规范，每一行是一个 glob 模式，用于指示应被忽略的文件。

::tip
你也可以在 `nuxt.config` 文件中配置 [`ignoreOptions`](/docs/4.x/api/nuxt-config#ignoreoptions)、[`ignorePrefix`](/docs/4.x/api/nuxt-config#ignoreprefix) 和 [`ignore`](/docs/4.x/api/nuxt-config#ignore)。
::

## 用法

```bash [.nuxtignore]
# 忽略布局 foo.vue
app/layouts/foo.vue
# 忽略名称以 -ignore.vue 结尾的布局文件
app/layouts/*-ignore.vue

# 忽略页面 bar.vue
app/pages/bar.vue
# 忽略 ignore 文件夹内的页面
app/pages/ignore/*.vue

# 忽略 foo 文件夹下的路由中间件文件，除了 foo/bar.js
app/middleware/foo/*.js
!app/middleware/foo/bar.js
```

关于该规范的更多细节，请参见 **gitignore 文档**。
