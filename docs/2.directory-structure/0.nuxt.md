::important
这个目录应该被添加到你的 [`.gitignore`](/docs/4.x/directory-structure/gitignore) 文件中，以避免将开发构建输出推送到你的代码仓库。
::

这个目录对于想要更多了解 Nuxt 根据你的目录结构生成的文件的人会很有意思。

Nuxt 还提供了一个虚拟文件系统（VFS），供模块向该目录添加模板而不需要将它们写入磁盘。

你可以在开发模式下打开 [Nuxt DevTools](https://devtools.nuxt.com)，并导航到 **Virtual Files** 选项卡来查看生成的文件。

::warning
你不应该修改其中的任何文件，因为在运行 [`nuxt dev`](/docs/4.x/api/commands/dev) 时，整个目录会被重新创建。
::
