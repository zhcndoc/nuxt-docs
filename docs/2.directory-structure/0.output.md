head.title: ".output/"
navigation.icon: i-vscode-icons-folder-type-package
---

::important
这个目录应当被添加到你的 [`.gitignore`](/docs/4.x/directory-structure/gitignore) 文件中，以避免将构建输出推送到你的仓库。
::

使用此目录将你的 Nuxt 应用部署到生产环境。

:read-more{to="/docs/4.x/getting-started/deployment"}

::warning
你不应该修改其中的任何文件，因为在运行 [`nuxt build`](/docs/4.x/api/commands/build) 时，整个目录会被重新创建。
::
