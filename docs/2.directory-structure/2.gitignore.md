`.gitignore` 文件指定 Git 应忽略的有意不被跟踪的文件。

:read-more{icon="i-simple-icons-git" title="Git 文档" to="https://git-scm.com/docs/gitignore" target="_blank"}

我们建议在 `.gitignore` 文件中**至少**包含以下条目：

```bash [.gitignore]
# Nuxt 开发/构建输出
.output
.data
.nuxt
.nitro
.cache
dist

# Node 依赖包
node_modules

# 日志文件
logs
*.log

# 其他杂项
.DS_Store

# 本地环境文件
.env
.env.*
!.env.example
```
