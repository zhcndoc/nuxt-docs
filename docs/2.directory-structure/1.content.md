[Nuxt Content](https://nuxt-content.zhcndoc.com) 会读取你项目中的 `content/` 目录 并解析 `.md`、`.yml`、`.csv` 和 `.json` 文件，为你的应用创建一个基于文件的 CMS。

- 使用内置组件渲染你的内容。
- 使用类 MongoDB 的 API 查询你的内容。
- 在 Markdown 文件中使用 MDC 语法插入 Vue 组件。
- 自动生成你的导航。

::read-more{to="https://nuxt-content.zhcndoc.com" target="_blank"}
在 **Nuxt Content** 文档中了解更多信息。
::

## 启用 Nuxt Content

通过一条命令在你的项目中安装 `@nuxt/content` 模块并将其添加到 `nuxt.config.ts`：

```bash [Terminal]
npx nuxt module add content
```

## 创建内容

将你的 Markdown 文件放到 `content/` 目录中：

```md [content/index.md]
# Hello Content
```

该模块会自动加载并解析它们。

## 渲染内容

要渲染内容页面，使用 [`<ContentRenderer>`](https://nuxt-content.zhcndoc.com/docs/components/content-renderer) 组件添加一个 [catch-all 路由](/docs/4.x/directory-structure/app/pages/#catch-all-route)：

```vue [app/pages/[...slug\\].vue]
<script lang="ts" setup>
const route = useRoute()
const { data: page } = await useAsyncData(route.path, () => {
  return queryCollection('content').path(route.path).first()
})
</script>

<template>
  <div>
    <header><!-- ... --></header>

    <ContentRenderer
      v-if="page"
      :value="page"
    />

    <footer><!-- ... --></footer>
  </div>
</template>
```
