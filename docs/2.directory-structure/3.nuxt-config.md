`nuxt.config` 文件的扩展名可以是 `.js`、`.ts` 或 `.mjs`。

```ts twoslash [nuxt.config.ts]
export default defineNuxtConfig({
  // 我的 Nuxt 配置
})
```

::tip
`defineNuxtConfig` 辅助函数全局可用，无需导入。
::

如果你愿意，也可以显式从 `nuxt/config` 导入 `defineNuxtConfig`：

```ts twoslash [nuxt.config.ts]
import { defineNuxtConfig } from 'nuxt/config'

export default defineNuxtConfig({
  // 我的 Nuxt 配置
})
```

::read-more{to="/docs/api/configuration/nuxt-config"}
在 **Nuxt 配置** 文档中了解所有可用的选项。
::

为了确保你的配置是最新的，当检测到主配置文件、[`.env`](/docs/3.x/directory-structure/env)、[`.nuxtignore`](/docs/3.x/directory-structure/nuxtignore) 和 [`.nuxtrc`](/docs/3.x/directory-structure/nuxtrc) 点文件发生变化时，Nuxt 会进行完整重启。
