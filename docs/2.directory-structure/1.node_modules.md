包管理器（[`npm`](https://docs.npmjs.com/cli/commands/npm/) 或 [`yarn`](https://yarnpkg.com) 或 [`pnpm`](https://pnpm.io/cli/install) 或 [`bun`](https://bun.com/package-manager)）会创建此目录来存储项目的依赖。

::important
该目录应添加到你的[`.gitignore`](/docs/3.x/directory-structure/gitignore) 文件中，以避免将依赖推送到你的代码仓库。
::
