---
title: 自动导入
description:  Nuxt 3 采用最小摩擦的方式，这意味着在可能的情况下组件和可组合项会被自动导入。
---

::note
在其余的迁移文档中，你会注意到关键的 Nuxt 和 Vue 工具没有显式导入。这不是笔误；Nuxt 会为你自动导入它们，如果你已按照[说明](/docs/4.x/migration/configuration#typescript) 使用 Nuxt 的 TypeScript 支持，你应该能获得完整的类型提示。
::

[了解有关自动导入的更多信息](/docs/4.x/guide/concepts/auto-imports)

## 迁移

1. 如果你在 Nuxt 2 中使用过 `@nuxt/components`，可以在你的 `nuxt.config` 中移除 `components: true`。如果你的配置更复杂，请注意组件选项已发生一些变化。有关更多信息，请参见[组件文档](/docs/4.x/guide/directory-structure/app/components)。

::tip
你可以查看 `.nuxt/types/components.d.ts` 和 `.nuxt/types/imports.d.ts`，以了解 Nuxt 如何解析你的组件和可组合的自动导入。
::
