---
title: 概述
description: Nuxt 3 是对 Nuxt 2 的全面重写，并且基于一套新的底层技术。
---

将 Nuxt 2 应用迁移到 Nuxt 3 时会有重大变化，不过随着我们朝稳定版本推进，迁移过程预计会变得更加简单。

::note
此迁移指南正在更新中，以配合 Nuxt 3 的开发。
::

其中一些重大变化包括：

1. 从 Vue 2 迁移到 Vue 3，包括默认使用组合式 API（Composition API）和 script setup。
1. 从 webpack 4 和 Babel 迁移到 Vite 或 webpack 5 和 esbuild。
1. 从运行时的 Nuxt 依赖转变为使用 nitropack 编译的精简独立服务器。

::tip
如果你需要继续使用 Nuxt 2，但想在 Nuxt 2 中受益于 Nuxt 3 的特性，也可以查看 [如何开始使用 Bridge](/docs/4.x/bridge/overview)。
::
