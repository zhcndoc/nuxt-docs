## ESLint

对于 Nuxt，推荐的做法是通过 [`@nuxt/eslint`](https://eslint.nuxt.com/packages/module) 模块启用 ESLint 支持，该模块会为你设置基于项目的 ESLint 配置。

:::callout{icon="i-lucide-lightbulb"}
该模块专为[新的 ESLint 扁平配置格式](https://eslint.org/docs/latest/use/configure/configuration-files)设计，这是[自 ESLint v9 起的默认格式](https://eslint.org/blog/2024/04/eslint-v9.0.0-released/)。如果你仍在使用传统的 `.eslintrc` 配置，则需要[使用 `@nuxt/eslint-config` 手动配置](https://eslint.nuxt.com/packages/config#customizing-the-config)。我们强烈建议你迁移到扁平配置，以便更好地适应未来的发展。
:::

## 快速设置

```bash
npx nuxt module add eslint
```

启动你的 Nuxt 应用后，项目根目录下会生成一个 `eslint.config.mjs` 文件。你可以根据需要对其进行自定义。

你可以在 [Nuxt ESLint 的文档](https://eslint.nuxt.com/packages/module) 中了解有关该模块和自定义的更多信息。
