---
title: Vite
description: '通过 Nuxt Bridge 在你的 Nuxt 2 应用中启用 Vite。'
---

::warning
使用 `vite` 时，必须配置好 [nitro](/docs/bridge/nitro)。
::

## 移除模块

- 移除 `nuxt-vite`：Bridge 已启用相同功能

## 更新配置

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  bridge: {
    vite: true,
    nitro: true
  }
})
```

## 配置

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  vite: {
    // Vite 的配置
  }
})
```
