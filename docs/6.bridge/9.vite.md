---
title: Vite
description: '通过 Nuxt Bridge 在你的 Nuxt 2 应用中启用 Vite。'
---

::warning
在使用 `vite` 时，必须先配置 [nitro](/docs/4.x/bridge/nitro)。
::

## 删除模块

- 删除 `nuxt-vite`：Bridge 提供相同的功能

## 更新配置

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  bridge: {
    vite: true,
    nitro: true,
  },
})
```

## 配置

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  vite: {
    // Vite 的配置
  },
})
```
