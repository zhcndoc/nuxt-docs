在 [Nuxt 上下文](/docs/guide/going-further/nuxt-app#the-nuxt-context) 中，你可以使用 `useRequestEvent` 来访问传入的请求。

```ts
// 获取底层请求事件
const event = useRequestEvent()

// 获取 URL
const url = event?.path
```

::tip
在浏览器中，`useRequestEvent` 会返回 `undefined`。
::
