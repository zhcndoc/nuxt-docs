::note
使用深度赋值更新 [`app.config`](/docs/guide/directory-structure/app-config)。现有的（嵌套）属性将被保留。
::

## 用法

```js
const appConfig = useAppConfig() // { foo: 'bar' }

const newAppConfig = { foo: 'baz' }

updateAppConfig(newAppConfig)

console.log(appConfig) // { foo: 'baz' }
```
