Nuxt 提供了一个日志实例，你可以使用它记录带有额外功能的消息。`useLogger` 允许你获取一个日志实例。

## `useLogger`

返回一个日志实例。它在底层使用 [consola](https://github.com/unjs/consola)。

### Usage

```ts twoslash
import { defineNuxtModule, useLogger } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    const logger = useLogger('my-module')

    logger.info('Hello from my module!')
  },
})
```

### Type

```ts
function useLogger (tag?: string, options?: Partial<ConsolaOptions>): ConsolaInstance
```

### Parameters

**`tag`**：用于在所有日志消息后添加的标签，显示在时间戳右侧。

**`options`**：Consola 的配置选项。

### Examples

```ts twoslash
import { defineNuxtModule, useLogger } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    const logger = useLogger('my-module', { level: options.quiet ? 0 : 3 })

    logger.info('Hello from my module!')
  },
})
```
