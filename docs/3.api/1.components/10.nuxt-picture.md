`<NuxtPicture>` 是原生 `<picture>` 标签的可替换组件，可直接替换原生标签。

`<NuxtPicture>` 的用法与 [`<NuxtImg>`](/docs/4.x/api/components/nuxt-img) 几乎相同，但它还允许在可能时提供诸如 `webp` 之类的现代格式。

在 MDN 上了解更多关于 [`<picture>` 标签](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/picture)。

## 设置

要使用 `<NuxtPicture>`，应安装并启用 Nuxt Image 模块：

```bash [Terminal]
npx nuxt module add image
```
